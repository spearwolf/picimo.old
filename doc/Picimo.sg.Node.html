readonlyappPicimo.App

The app instance

displayboolean

If set to false the node won't be rendered. The frame, renderFrame and frameEnd events won't be emitted. BUT initialization will be happen. (If you don't want the node to initialize set the ready attribute to false).

readonlyisRootPicimo.sg.Node

True if this is the root node of the scene graph.

readyboolean

A node ist not ready if ..

  1. the state is set to destroyed or error
  2. the parent scene is set or if not the isRoot flag is set to true
  3. you explicitly set it to false (but default is true)
